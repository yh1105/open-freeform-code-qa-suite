The `myprogram-0.0.1-plain.jar` file is generated by the `bootJar` task of the Spring Boot Gradle plugin. This is a feature introduced in Spring Boot 2.3.0 to support layered jars.

If you want to disable the generation of the `*-plain.jar` file, you can configure the `bootJar` task in your `build.gradle` file to disable the `enabled` property of the `plain` archive:

```groovy
bootJar {
    archiveFileName = 'myprogram-0.0.1.jar'
    enabled = true
    tasks.named('bootJar').configure {
        enabled = true
    }
    tasks.named('jar').configure {
        enabled = false
    }
}
```

This will only generate the `myprogram-0.0.1.jar` file.
