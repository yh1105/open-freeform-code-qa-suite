The issue you're facing is due to the default behavior of the Scaffold widget in Flutter. When the keyboard appears, the Scaffold widget moves its body and bottom navigation bar up to avoid being covered by the keyboard.

To disable this behavior, you can use the `resizeToAvoidBottomInset` property of the Scaffold widget and set it to `false`. This will prevent the Scaffold from resizing when the keyboard appears.

Here's how you can do it:

```dart
class _AppHomeViewState extends State<AppHomeView>
    with TickerProviderStateMixin {

  TabController tabController;

  @override
  void initState() {
    super.initState();
    tabController = TabController(length: 4, vsync: this, initialIndex: 0);
    tabController.addListener(handleTabSelection);
  }

  @override
  Widget build(BuildContext context) {
    final scaffold = Scaffold(
      resizeToAvoidBottomInset: false, // Add this line
      body: SafeArea(child: _buildBody(context)),
      bottomNavigationBar: Container(
        height: 48,
        decoration: BoxDecoration(
          color: StyledColors.BACKGROUND_COLOR,
          boxShadow: [
            BoxShadow(
              color: StyledColors.FORGROUND_COLOR.withOpacity(0.16),
              blurRadius: 12,
              offset: Offset(0, 0),
            ),
          ],
        ),
        child: SafeArea(
          child: _buildTabBar(context),
        ),
      ),
    );
  }
  ...
}
```

This will make the bottom navigation bar stay behind the keyboard when it appears.
