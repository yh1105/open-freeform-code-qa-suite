You can use the Tailwind CSS utility classes to apply a border to the top edge only. You should use `border-t` for top border and specify the color with `border-blue-900`. Here's how you can do it:

```html
<div class="border-t border-blue-900">foo</div>
```

This will apply a border only to the top edge of the div. The `border-t` class applies a border to the top edge, and `border-blue-900` sets the color of the border.
