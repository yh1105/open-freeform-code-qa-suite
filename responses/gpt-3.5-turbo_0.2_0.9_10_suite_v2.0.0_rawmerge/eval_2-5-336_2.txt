Here's the implementation of the `createPriorityQueue` function in Java:

```java
public static PriorityQueue<Pair<Integer, Integer>> createPriorityQueue() {
    return new PriorityQueue<>((a, b) -> a.getKey() - b.getKey());
}
```

This function creates a new `PriorityQueue` of `Pair<Integer, Integer>` objects, where the smallest key of pairs is considered as priority. The lambda expression `(a, b) -> a.getKey() - b.getKey()` is used to compare the pairs based on their keys. The `getKey()` method is used to retrieve the key of each pair.
