You can use the `resizeToAvoidBottomInset` property of the `Scaffold` widget to disable the behavior of the bottom navigation bar moving up with the keyboard. Set it to `false` to prevent the `Scaffold` from resizing when the keyboard appears. Here's an example:

```dart
class _AppHomeViewState extends State<AppHomeView>
    with TickerProviderStateMixin {

  TabController tabController;

  @override
  void initState() {
    super.initState();
    tabController = TabController(length: 4, vsync: this, initialIndex: 0);
    tabController.addListener(handleTabSelection);
  }

  @override
  Widget build(BuildContext context) {
    final scaffold = Scaffold(
      resizeToAvoidBottomInset: false, // disable resizing when keyboard appears
      body: SafeArea(child: _buildBody(context)),
      bottomNavigationBar: Container(
        height: 48,
        decoration: BoxDecoration(
          color: StyledColors.BACKGROUND_COLOR,
          boxShadow: [
            BoxShadow(
              color: StyledColors.FORGROUND_COLOR.withOpacity(0.16),
              blurRadius: 12,
              offset: Offset(0, 0),
            ),
          ],
        ),
        child: SafeArea(
          child: _buildTabBar(context),
        ),
      ),
    );
  }
  ...
}
``` 

By setting `resizeToAvoidBottomInset` to `false`, the `Scaffold` will not resize when the keyboard appears, and the bottom navigation bar will stay behind the keyboard.
