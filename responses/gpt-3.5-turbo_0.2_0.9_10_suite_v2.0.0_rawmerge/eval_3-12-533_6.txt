You can wrap your `Scaffold` with a `GestureDetector` and listen to its `onTap` event to detect a tap on the screen. Here's an example:

```dart
class QQHome extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      theme: ThemeData(
        primaryColor: Colors.blueGrey,
      ),
      home: GestureDetector(
        onTap: () => print('Tapped'),
        child: Scaffold(
          appBar: AppBar(
            centerTitle: true,
            title: Text('QuoteQuota'),
          ),
          body: QQBody(),
        ),
      ),
    );
  }
}

class QQBody extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Center(
      child: Text(
        'Hello, World!'
      ),
    );
  }
}
```

This way, the `GestureDetector` will detect a tap on the screen and trigger the `onTap` event, regardless of where the tap occurs within the `Scaffold`.
