You can use the OneDrive API to interact with OneDrive using VBA. You can find more information about the OneDrive API here: https://docs.microsoft.com/en-us/onedrive/developer/rest-api/getting-started/?view=odsp-graph-online. Additionally, you can use the SharePoint REST API to interact with SharePoint using VBA. You can find more information about the SharePoint REST API here: https://docs.microsoft.com/en-us/sharepoint/dev/sp-add-ins/get-to-know-the-sharepoint-rest-service?tabs=csom. As for retrieving the local path of the file, you can use the following VBA code: `ThisWorkbook.Path`. This will return the local path of the current workbook.
