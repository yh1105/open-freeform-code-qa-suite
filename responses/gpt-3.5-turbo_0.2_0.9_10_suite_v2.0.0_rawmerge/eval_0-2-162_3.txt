You can change the background color dynamically on scroll by adding the following code:

```javascript
const [red, green, blue] = [69, 111, 225]
const section1 = document.querySelector('.section1')

window.addEventListener('scroll', () => {
  let y = 1 + (window.scrollY || window.pageYOffset) / 150
  y = y < 1 ? 1 : y // ensure y is always >= 1 (due to Safari's elastic scroll)
  const [r, g, b] = [red/y, green/y, blue/y].map(Math.round)
  section1.style.backgroundColor = `rgb(${r}, ${g}, ${b})`
})
```

This code listens for the `scroll` event on the window object and calculates the current scroll position. It then uses this position to adjust the background color of the `.section1` element dynamically.
