You can wrap your `Scaffold` with a `GestureDetector` instead of wrapping just the `QQBody` widget. This will allow you to detect taps anywhere on the screen. Here's an example:

```dart
class QQHome extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return MaterialApp(
      debugShowCheckedModeBanner: false,
      theme: ThemeData(
        primaryColor: Colors.blueGrey,
      ),
      home: GestureDetector(
        onTap: () => print('Tapped'),
        child: Scaffold(
          appBar: AppBar(
            centerTitle: true,
            title: Text('QuoteQuota'),
          ),
          body: QQBody(),
        ),
      ),
    );
  }
}

class QQBody extends StatelessWidget {
  @override
  Widget build(BuildContext context) {
    return Center(
      child: Text(
        'Hello, World!'
      ),
    );
  }
}
```

Now, when you tap anywhere on the screen, the `onTap` function of the `GestureDetector` will be triggered and "Tapped" will be printed to the console.
