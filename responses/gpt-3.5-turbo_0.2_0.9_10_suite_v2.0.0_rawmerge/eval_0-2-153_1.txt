You can use the `border-t` class instead of `border` to apply the border only to the top of the element. Here's an example:

```html
<div class="border-t border-blue-900 border-t-1">foo</div>
```

This will apply a border of width 1 to the top edge of the element, with a blue color. The `border-t` class specifies that the border should only be applied to the top edge of the element.
