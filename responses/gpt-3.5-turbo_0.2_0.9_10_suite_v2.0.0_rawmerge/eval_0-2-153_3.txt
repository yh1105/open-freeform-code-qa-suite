You can use the `border-t` class instead of `border` to draw a border only on the top edge of the element. Here's an example:

```html
<div class="border-t border-blue-900 border-t-1">foo</div>
```

This will draw a 1-pixel wide blue border only on the top edge of the `div` element. The `border-t` class is a shorthand for `border-top-width: 1px; border-top-style: solid;`.
