You can use the `conda list --show-channel-urls` command to display the package size in bytes. Here's how to do it:

1. Open the Anaconda Prompt or your terminal.
2. Type `conda list --show-channel-urls` and press Enter.
3. Wait for the list of packages to be displayed.
4. Look for the `size` column. This column shows the size of each package in bytes.

Note that the `--show-channel-urls` option is required to display the `size` column. Without this option, the `size` column will not be displayed.
